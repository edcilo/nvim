## Install neovim

### Linux

```bash
apt install neovim
```

### MacOs

```bash
brew install neovim
```

## Dependencies

```bash
curl
git
nodejs
npm
python3
python3-pip
unzip
wget
zip
```

## Clone repositorie

```bash
git clone https://github.com/edcilo/nvim.git ~/.config/nvim
```

## Providers

Configure path of providers in `~/.config/nvim/lua/providers`

## Install plugins

From vim in normal mode

```bash
:PackerInstall
```

Check health

```bash
:checkhealth
```
